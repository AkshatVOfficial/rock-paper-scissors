# 🎮 Rock Paper Scissors Game

A modern and responsive Rock Paper Scissors game built using **HTML**, **CSS**, and **Vanilla JavaScript**.

Challenge the computer, track scores, and enjoy a clean user interface with custom icons and animations.

## 🎮 How to Play

1. Click on Rock, Paper, or Scissors.
2. The computer randomly selects its move.
3. The winner is decided based on classic rules:

```text
Rock beats Scissors
Scissors beats Paper
Paper beats Rock
```

4. Scores update automatically after each round.
5. Use the Reset button to start a new game.
